AVR Assembly Programming

An example-driven book on writing AVR assembly with the GNU toolchain — avr-gcc / avr-as — targeting the ATtiny3217 Curiosity Nano. Every topic ships with runnable code.

⬇ Download the PDF View on GitHub

What the book covers

  • AVR architecture: register file, SREG, stack, memory map
  • GAS syntax, directives, sections, macros, and objdump
  • The instruction set: operands, flags, branches, skips, the ABI
  • Load/store addressing, pointer registers, flash & SRAM access
  • Multi-byte arithmetic, multiply/divide, fixed-point, BCD, CORDIC
  • GPIO, interrupts, clock/power control
  • Timers, USART, SPI, TWI/I²C, ADC
  • UPDI programming and bootloader self-programming
  • GDB debugging, cycle counting, optimization, linker scripts
  • Appendices: registers, ISA, GAS directives, avrdude workflows

Build it yourself

Clone the repo and build the combined PDF with pandoc + XeLaTeX:

git clone https://github.com/garyPenhook/AVR-assembly-with-avr-gcc-book.git
cd AVR-assembly-with-avr-gcc-book && make pdf
